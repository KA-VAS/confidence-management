# Conference UI Starter

A minimal React + Vite + TypeScript + Tailwind starter implementing a modern Academic Conference Management UI.

Quick start:

```bash
npm install
npm run dev
```

Routes included:
- `/` Landing
- `/dashboard` User dashboard
- `/admin` Admin dashboard
- `/submit` Paper submission
- `/reviews` Peer review panel
- `/schedule` Schedule
- `/notifications` Notifications
- `/profile` Profile

Design notes: light theme, blue-purple accents, card-based components, responsive.
